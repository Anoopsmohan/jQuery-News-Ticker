Cool news ticker! Seems familiar...

Taking inspiration from the BBC News website ticker, jQuery News Ticker brings a lightweight and easy to use news ticker to jQuery.

So how do I use this thing?!

If you're reading this then you've successfully downloaded the zip file containing the plugin (both development and minified versions) and this example / documentation - well done!

And what do I do with these files?

If you want the fully styled, development version (as above) of the news ticker, use the files from the development folder in the zip and include the following in your HTML:

<link href="css/style.css" rel="stylesheet" type="text/css" />
<script src="jquery.ticker.js" type="text/javascript"></script>

If you want just the minified, production version of the news ticker, use the files from the minified folder in the zip and include the following in your HTML:

<link href="css/min-style.css" rel="stylesheet" type="text/css" />
<script src="jquery.ticker.min.js" type="text/javascript"></script>

Unsurprisingly you will need to include jQuery (1.3.2 or above) to make all this work, there is a version (1.4.2) included in the zip.

Apologies to anyone using an earlier version, I may update the code be compatible with jQuery 1.3 if there is enough demand. There is now a version of jQuery news ticker included in the zip that is fully jQuery 1.3.2 compatible!

Don't I need some HTML to make this news ticker work?

Yes you do, all you need is the following HTML:

<div id="ticker-wrapper" class="no-js">
	<ul id="js-news" class="js-hidden">
		<li class="news-item"><a href="#">This is the 1st latest news item.</a></li>
		<li class="news-item"><a href="#">This is the 2nd latest news item.</a></li>
		<li class="news-item"><a href="#">This is the 3rd latest news item.</a></li>
		<li class="news-item"><a href="#">This is the 4th latest news item.</a></li>
	</ul>
</div>

And how do I make jQuery do it's magic on this thing?

To get jQuery News Ticker working, you'll need to also add this code to your page:

<script type="text/javascript">
	$(function () {
		$('#js-news').ticker();
	});
</script>

Ok, what else?

That's everything to get the news ticker working! If you get stuck have a look at the source of the example on this page to see what's what.

You'll probably want to dive into the CSS to change the visual style to your liking, but you're on your own with that one!

Ok, that's great and everything but I want to fiddle with the settings...

Ok, so here's what you can do an the moment:

speed: 0.10
The speed of the reveal
ajaxFeed: false
Populate the news items via a remote feed
feedUrl: false
The remote URL to populate the feed from, must be on the same domain as the ticker
feedType: 'xml'
The type of feed that the ticker should expect - currently only XML
htmlFeed: true
Populate the news items via the HTML
debugMode: true
Show some (possibly) helpful errors in the console or as alerts.
SHOULD BE SET TO FALSE FOR PRODUCTION SITES!
controls: true
Whether or not to show the tickers controls
titleText: 'Latest'
The "title" for the ticker
displayType: 'reveal'
The type of display for the ticker - current options are 'reveal' or 'fade'.
pauseOnItems: 2000
The amount of time a news item stays on the page before being replaced
fadeInSpeed: 300
Speed of a news items fade in animation
fadeOutSpeed: 300
Speed of a news items fade out animation

Woah, that made no sense, how do I use these settings?

Any changes you want to make to the default settings are passed in to the call of the news ticker, like so:

<script type="text/javascript">
	$(function () {
		$('#js-news').ticker({
			speed: 0.10,
			htmlFeed: false,
			fadeInSpeed: 600,
			titleText: 'Latest News'
		});
	});
</script>

Which browsers does this work with?

So far this plugin had been tested as working with: IE6+, FF 3.6+, Chrome, Safari, Safari Mobile.

If you do find any bugs, or any further browser compatibility, let me know.

What about loading content from a remote feed?!

Yeah, well the population of news items will be from HTML only for now I'm afraid...

You're in luck! The latest update to jQuery News Ticker introduced support for loading remote content via RSS!

Be warned, this is a very simple implementation and for it to work your XML feed should match the example found here: http://en.wikipedia.org/wiki/RSS#Example.

If you are integrating jQuery News Ticker into WordPress or similar you should be fine.

Further support for loading new items from a remote URL (HTML/JSON) will be coming in a future release, so stay tuned!
